# Realm 视觉语言

## 主题

post-apocalyptic HUD：旧式 CRT 扫描感、低饱和的琥珀 + 蓝灰、过扫描式留白、字阶强烈、装饰几何感。

## 字阶

- 标题：等宽 + 几何 sans 双线，h1 用 `clamp(2.4rem, 4vw, 3.2rem)`
- 正文：等宽优先；CJK 走 `Noto Sans Mono CJK`
- 数字 / 元信息：tabular-nums 等宽数字
- 翻译内容**不**使用 `Zelda Free` 等 Latin-only 字体，否则 CJK/带音字符会丢失

## 色板

- 背景：`#0a0c10`（主）/ `#10141b`（次）
- 主色：`#ffb454`（琥珀）
- 次色：`#5db9ff`（蓝灰）
- 警示：`#ff5b5b`
- 中性文字：`#c2c8d2` / `#7a8390`

## 间距与栅格

- 8pt 基准；常用间距 8 / 16 / 24 / 40 / 64
- 容器最大宽度 1200；正文区最大 720

## 关键组件

- HUD 左面板：固定栏，含站点状态、当前时间、提示
- 路由过渡：CRT 扫描线 + 噪声层 + 字符随机
- 受保护博文解锁：扫描动画 + 字符解码
- 媒体播放器：紧凑控制条 + 频谱条
